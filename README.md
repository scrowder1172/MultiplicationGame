# Challenge 03: Multiplication Game

## Hacking With Swift challenge 03

This is Challenge 03 in Paul Hudson's 100 Days of SwiftUI course. The project creates an iOS app that kids can use to learn their multiplication tables.

## Course Link

Check out Paul's course for more on SwiftUI: https://www.hackingwithswift.com/100/swiftui

## How to install

1. fork the project or download the .zip
2. open project in XCode v15 or later (earlier versions may or may not compile correctly)
3. run in simulator
